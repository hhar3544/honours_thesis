# honours_thesis

This repository contains custom Python scripts used for behavioural metric extraction, visualisation, and statistical proof-of-concept analysis in an honours thesis project investigating automated behavioural quantification of the mechanical conflict-avoidance (MCA) assay in mice.

## Project overview

The mechanical conflict-avoidance assay is a three-chamber behavioural paradigm used to assess pain-related avoidance behaviour. This project developed an automated analysis workflow that combines video preprocessing, pose estimation, ROI-based behavioural extraction, custom Python-based cleaning, metric calculation, visualisation, and statistical analysis.

The scripts in this repository were used to support the thesis:

**The development of a deep learning protocol for assessing pain modulated behaviour in mice mechanical conflict avoidance assay**
